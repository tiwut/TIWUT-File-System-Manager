# TIWUT File System Manager (Nexus Edition)

Welcome to the **TIWUT File System Manager**, completely rewritten and optimized for the Nexus ` programming language. This application allows you to manage, create, and interact with `.tiwut` files (specialized file system archives) directly through a Zenity-powered GUI dashboard.

## Features

- **Create New**: Initialize a new `.tiwut` file system complete with the required folder structure (`data/`, `info/`, `storage/`, `user/`, `meta/`, `app/`, `filesystem/`, `temp/`).
- **Open**: Load an existing `.tiwut` file into the manager.
- **Read & Edit**: Quickly preview and modify the contents of files.
- **Import**: Seamlessly import external files into specific directories of your `.tiwut` structure.
- **System Status**: View your current OS, User, System Info, and current Timestamp via the built-in Nexus system module.

## How to Run

To run this application, you will need the `nexus` interpreter.

1. Open your terminal.
2. Execute the file using your compiled Nexus interpreter. For example:
   ```bash
   ./nexus/interpreter app/main.nx
   ```
   *(Ensure that the paths to your interpreter and the `main.nx` file are correct for your current working directory).*

## 🚀 Available in Tiwut Launcher

Great news! You don't have to launch this from the command line manually. **This application is also available and fully integrated within the Tiwut Launcher!** 

Simply open your Tiwut Launcher, search for the *TIWUT File System Manager*, and launch it with a single click!
